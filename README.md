# CRUD-Student

## How to run application

* Install node_modules
```
npm install
```

* Run app
```
nodemon app
```

## API Querries

* Get all students
```
Method: GET
http://localhost:3000/students
```

* Get student by id
```
Method: GET
http://localhost:3000/student/id
```

* Create student
```
Method: POST
http://localhost:3000/student
```

* Update student by id
```
Method: PUT
http://localhost:3000/student/id
```

* Delete student by id
```
Method: DELETE
http://localhost:3000/student/id
```
